<功能介绍>

1.首页功能:
     轮播图,
     三级菜单,
     楼层效果,
     json数据渲染商品图片(共展示12种商品,共54张图片)

2.商品列表功能:
     选项卡,(banner)
     json数据渲染商品列表图片(共展示8种商品,共48张图片)

3.商品详情功能:
     放大镜,
     json数据渲染商品详情图片(共展示8种商品,共48张图片)
     

<跳转和效果介绍>
1.首页
    1)点击楼层下任意一件商品进入商品列表页
    2)点击左侧楼层可跳转到对应的楼层商品
    3)点击登录可跳转登录页面
    4)点击右侧"Top"可立即回到首页头部
    5)登录后"登录"和"注册"变成"已登录"和"退出",点击退出提示已退出,3秒后跳转登录页面
    6)点击右上角购物车,当没有注册会弹出提示框并在3秒后跳转注册页面
    7)点击右上角购物车,当注册没登录时会弹出提示框提示登录并在3秒后跳转登录页面
    8)点击右上角购物车,当登录后会立即跳转到购物车页面
    9)滑动全部商品分类,会出现三级餐单

2.商品列表
    1)滑过任意渲染商品,出现"加入购物车"选项
    2)点击"加入购物车"商品会加入到购物车中,右上角显示选择商品的总价
    3)点击右上角购物车会跳转到购物车页面
    4)点击左上角logo可以跳转到首页
    5)登录后"登录"和"注册"变成"已登录"和"退出",点击退出提示已退出,3秒后跳转登录页面
    6)点击右上角购物车,当没有注册会弹出提示框并在3秒后跳转注册页面
    7)点击右上角购物车,当注册没登录时会弹出提示框提示登录并在3秒后跳转登录页面
    8)点击右上角购物车,当登录后会立即跳转到购物车页面
    
3.商品详情页
    1)滑过商品主页面会出现放大镜效果
    2)点击左上角logo可以跳转到首页
    3)点击购物车会跳转到购物页面
    4)登录后"登录"和"注册"变成"已登录"和"退出",点击退出提示已退出,3秒后跳转登录页面
    5)点击右上角购物车,当没有注册会弹出提示框并在3秒后跳转注册页面
    6)点击右上角购物车,当注册没登录时会弹出提示框提示登录并在3秒后跳转登录页面
    7)点击右上角购物车,当登录后会立即跳转到购物车页面

4.购物车页面
    1)点击购买数量的"+","-"号,可以更改该商品的小计,并且"规格"和"总价"会实时更改
    2)手动输入更改购买时数量,可以更改该商品的小计和"规格"
    3)点击"删除"可以删除该类商品
    4)点击"继续购物"可以跳转到商品列表继续添加商品
    5)点击左上角logo可以跳转到首页
    6)登录后"登录"和"注册"变成"已登录"和"退出",点击退出提示已退出,3秒后跳转登录页面

5.注册页面
    1)点击右上角"登录"可以跳转到登录页面
    2)点击左上角logo可以跳转到首页
    3)手机号必须是11位数字并且第一位是"1",第二位是"3-9",并且是必填项
    4)两次密码是输入必须相同否则会报错,且不能小于4位
    5)输入相同的手机号输出会提示该"手机号已注册"
    5)其他选择均未为判断
    6)注册成功后会在二秒后跳转到登录页面

6.登录页面
    1)输入未注册手机号或密码错误时会提示"账号密码不符"
    2)没有填写验证码时会提示"请输入验证码"
    3)登录成功后会提示"登录成功"在两秒后跳转到首页
    4)点击左上角logo可以直接跳转到首页
    5)点击"立即注册"可以跳转到注册页面
      
<特点>
    1)页面均用json数据宣传商品
    2)json采用多重二维数组,实现渲染商品多样化存放
    3)购物车总价与点击数量实时更改
    4)账号登录后会在所有页面显示已登录并且可以随时退出
    5)根据"未注册""未登录""已登录"状态判断点击购物车跳转状态

<缺点>
    1)分页,模块化为实现
    2)不能存储多账号购物车状态
    3)未做浏览器版本兼容
